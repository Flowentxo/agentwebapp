## ğŸ¯ Key Interactions

### Drag and Drop
1. **From Palette to Canvas:** Drag node from palette, drop on canvas
2. **Node to Node:** Drag from output handle to input handle to create edge
3. **Parameter Mapping:** Drag from source field to target field

### Configuration
1. **Click Node:** Opens config panel on right
2. **Double-click Node:** Opens config panel in modal (alternative)
3. **Right-click Node:** Shows context menu (Edit, Duplicate, Delete, Test)

### Testing
1. **Click "Test" button:** Opens test interface inline
2. **Fill parameters:** Auto-generated form based on parameter schema
3. **Execute:** Sends request to backend
4. **View results:** Shows formatted results with timing info
